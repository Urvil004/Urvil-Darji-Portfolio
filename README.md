## ✨ About This Portfolio

A high-performance personal portfolio built with **React**, **TypeScript**, **GSAP**, **Three.js**, and **WebGL** — originally designed by [Urvil Darji](https://urvil-portfolio.vercel.app/) and fully transformed into my own.

The portfolio features:
- 🎭 **3D interactive character** powered by Three.js & WebGL
- ⚡ **GSAP scroll animations** with smooth scroll triggers
- 🖱️ **Custom cursor** with magnetic hover effects
- 📱 **Fully responsive** across all devices
- 🌙 **Dark theme** with teal accent design system

## ⚙️ Getting Started

### Prerequisites
- Node.js `v18+`
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Urvil004/Urvil-Darji-Portfolio.git

# Navigate into the project
cd Urvil-Darji-Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
